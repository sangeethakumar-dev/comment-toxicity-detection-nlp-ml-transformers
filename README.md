# Comment Toxicity Detection using NLP, Machine Learning, and Transformers

## Project Overview
This project focuses on detecting toxic comments using Natural Language Processing (NLP), traditional Machine Learning models, and Transformer-based models. The system classifies whether a comment is toxic or non-toxic and aims to support safer online communication and content moderation.

The project also includes a Streamlit web application for real-time toxicity prediction.

---

## Objectives
- Perform text preprocessing using NLP techniques
- Convert text into numerical features using TF-IDF
- Train and evaluate multiple Machine Learning models
- Experiment with Transformer-based models such as BERT
- Build an interactive Streamlit application
- Deploy the trained model for real-time predictions
